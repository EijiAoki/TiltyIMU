bool ClassHTMC::StartCal ( )

Start the calibration. The sensor should be rotated a little more than 360 along the horizontal plane in no less than 20 seconds. After the sensor has been rotated, call StopCal() to set the sensor back in measurement mode and save the calibration data. This calibration data is stored in the sensor until the next calibration.

Returns:
true if no error occured, otherwise false
Examples:
Example_HiTechnic_Compass_Calibrate.ino.

bool ClassHTMC::StopCal ( )
